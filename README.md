# Arduino-parking-system-in-ASTU
ASTU Smart Parking System  . An Arduino-based gate controller for parking buildings. Uses an MFRC522 RFID scanner to verify UIDs and a Servo motor for the barrier.  Features:  Access Control: Only authorized IDs open the gate.  Capacity Logic: Tracks car count; denies entry when full.  Simulation: Built &amp; tested in Wokwi.  Built by Eyob.
